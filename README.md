- 🤖 **Agents** — Tool-using autonomous agents

- 🛠️ **Custom Tools** — Building LangChain tools

## 🚀 Setup

```bash
git clone https://github.com/aasimansari1/LangChain-Models.git
cd LangChain-Models
pip install -r requirements.txt
# Add your OPENAI_API_KEY to .env
python main.py
```
